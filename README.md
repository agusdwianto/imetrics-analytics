## üöÄ Installation Guide for Journal Managers
Integrating I-Metrics into your journal sidebar is simple:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/agusdwianto/imetrics-analytics.git](https://github.com/agusdwianto/imetrics-analytics.git)
    ```
2.  **Add Custom Block:** In your OJS dashboard, navigate to *Settings > Website > Plugins > Custom Block Manager*.
3.  **Embed Code:** Copy the structure from `index.html` and link the `script.js` hosted on your domain or GitHub Pages.
4.  **Auto-Detection:** The script automatically detects the **DOI** from your article metadata to populate the data.
